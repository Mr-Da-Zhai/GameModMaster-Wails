## 📝 项目介绍

GameModMaster-Wails 是一个风灵月影游戏修改器管理工具，使用 Wails v3 + Go + Vue 3 开发。提供高效的修改器浏览、搜索、下载和管理功能。

### ✨ 主要特性

- 🎮 修改器浏览与搜索（支持中英文搜索）
- 📥 一键下载安装修改器
- 🚀 离线优先，秒开首页
- 📊 高效表格视图，信息密度高
- 🔄 自动检测更新
- 🌐 多语言支持

## 🛠️ 技术栈

- [Wails v3](https://wails.io/) - Go + WebView 桌面框架
- [Go](https://go.dev/) - 后端逻辑
- [Vue 3](https://vuejs.org/) - 前端框架
- [TypeScript](https://www.typescriptlang.org/) - 类型安全
- [Naive UI](https://www.naiveui.com/) - UI 组件库
- [SQLite](https://www.sqlite.org/) - 本地数据持久化

## 📦 开发

```bash
# 安装前端依赖
cd frontend && pnpm install

# 开发模式
wails3 dev

# 构建
wails3 build
```
